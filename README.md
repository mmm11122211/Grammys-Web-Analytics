# Analyzing Website Performance for The Grammys

## Project Overview
This project analyzes real web traffic data from The Recording Academy ("the Grammys") to measure the impact of splitting their digital presence into two separate websites: grammy.com and recordingacademy.com.

## Technologies Used
- **Python**: Data manipulation and analysis.
- **Pandas**: For processing `grammys_live_web_analytics.csv` and `ra_live_web_analytics.csv`.
- **Plotly Express**: For generating interactive visualizations of visitor trends.

## Key Insights
- Analyzed visitor traffic, bounce rates, and session duration.
- Compared engagement metrics during "Awards Week" vs. standard days.
